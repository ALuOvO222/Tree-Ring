音乐文件夹 / Music Folder
==================================================

请将音乐文件放在此文件夹中以启用音乐同步功能。
Please put music files in this folder to enable music sync.

支持的格式 / Supported formats:
- MP3 (.mp3)
- WAV (.wav)  
- OGG (.ogg)

推荐音乐类型 / Recommended music types:
- 自然环境音 / Nature ambient sounds
  * 森林声音 / Forest sounds
  * 雨声 / Rain sounds  
  * 鸟鸣 / Bird songs
  * 流水声 / Water flowing

- 古典音乐 / Classical music
  * 钢琴独奏 / Piano solo
  * 弦乐四重奏 / String quartet
  * 巴洛克音乐 / Baroque music

- 新世纪音乐 / New Age music
  * 冥想音乐 / Meditation music
  * 治愈音乐 / Healing music
  * 环境音乐 / Ambient music

- 缓慢节奏的器乐 / Slow-tempo instrumental
  * 民谣吉他 / Folk guitar
  * 大提琴 / Cello
  * 长笛 / Flute

音乐同步功能 / Music Sync Features:
==================================================
- 动画速度会根据音乐节拍调整
- 年轮生长会跟随音乐的强弱变化
- 不同频率的音乐会影响视觉效果
- 支持实时音频分析和节拍检测

如果没有音乐文件，程序将使用合成节拍。
If no music files are found, synthetic beats will be used.

建议文件大小 / Recommended file size:
- 单个文件不超过50MB
- 总音乐文件不超过200MB  
- 采样率：44.1kHz 或 48kHz
- 比特率：128kbps 或更高

注意事项 / Notes:
==================================================
1. 请确保音乐文件版权合法
2. 程序支持循环播放
3. 可以通过键盘控制音乐播放
4. 音乐音量可以实时调节
5. 支持多首音乐自动切换

快速测试 / Quick Test:
==================================================
1. 放入一首音乐文件
2. 运行: python demo.py 或 python main.py
3. 按 M 键切换音乐开关
4. 观察动画与音乐的同步效果

享受音乐与自然的完美结合！
Enjoy the perfect combination of music and nature!
